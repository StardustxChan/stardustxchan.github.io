My Hackintosh in 2016

为什么写这篇文

  1. 配件选择有诸多取舍, 具体罗列每一种方案来相互比较, 是写这个文章的起因.
  2. 少说玄学, 集中在这几个问题上. “哪几样好, 这个用起来怎么样, 我该怎么选” .
  3. 因此我会很详细的描述每一样配件的选择和驱动情况.
  4. 不是教程帖，也没能力做教程。爬帖学习是最快最靠谱的

  5. 浅谈思路分享, 以及踩过的坑.

发这篇文章的时候, 主机已经二手卖出了, 最终还是用回了笔记本.
笔记本是 神舟K640e, 等同于 键盘面是黑色 的蓝天模具的 K610d4 和 K650, 笔记本或台式如果和我配置一样, 直接问我要文件就可以最无痛的装好系统了.

Hackintosh_K610D4_EFI, 这里下载

为什么没买白苹果

  1. 等了半年没发布 Retina MacBook Pro, 过了冲动消费期
    1. 16年15寸起价18k, 配置感人
    2. 核显15寸14k的售价 = 整台黑苹果 + 8.xk,
    3. 8.xk = 二手13年Retina MacBook Pro15 or 港版Retina MacBook Pro13
  2. 新 Retina MacBook Pro 槽点实在太多, 配置中没有几样是满意的.
  3. 倾向于拥有一套可改动的配置, 而非一站式解决方案
  4. 既然是省钱驱动的装机, 唯一能买的白果 Mac Mini 就不在考虑范围内了

配置

平台 GA-B150M-D3H + i7-6700

B150 这种直接选一线板, 技嘉B150M D3H, 就是接口够, 然后便宜. PCIe 的 M.2 口, 和看起来很好驱动的 ALC892, 买的人也多, 配置可以参考(然而最终还是去看文档了)

按理说技嘉对黑苹果支持比较好, 然而这一代却不支持原生电源管理了. 这就很尴尬了

事实上，不买华擎真是脑子进水，考虑到非k超外频, B150 无论从性能还是价格, 都推荐买 华擎Hyper Pro4.

6700 这个温度是真的服, 轻度使用才 30 度, 我估计都没我的 SSD 温度高. 这里只读到机械硬盘的温度.

###SSD SM951 -NVMe

这个爆款玩意能用上的确是好东西, 老主板连 M.2 接口都没有, 不过可以转接到 x16 显卡位上.因为一般的 CPU 就16条通道, 一般放一张显卡. 不过千元显卡 x8 足够, 所以尝试把显卡插 x8, SSD 插第一槽x16. 这样可以保证 SSD 的速度. 不过也有人说, 第一槽显卡, 第二槽 SSD, 显卡会抢 SSD 带宽, 蜜汁问题. 没机会遇到了.

NVMe 是必须要有的, 直到2016款 Retina MacBook Pro, 苹果才有自己的 NVMe 设备. 所以在 10.11, 驱动是把盘作为 PCI 外挂盘驱动来曲线救国, 到了10.12, macOS 原生驱动, 只需在 config 里加个补丁就好了. (能用就好了)

然而开关机依然很慢, 哈哈.

购买选择方面, 只推荐实惠且好用的.

  1. 到了现在, 建兴T10上市, 我买的这些个拆机盘都降价的厉害, 保修与速度, 请自行抉择.
  2. 先看看 M.2 能不能走 PCIe 通道, 支不支持 NVMe 启动, 否则要刷 BIOS
  3. 再看看是不是在 PCIe 3.0 x4 以上, 否则会限速
  4. SM951 的 AHCI 版本就不要买了, 游戏玩家上 Intel 600P , 发烧友就 SM961
  5. 发热方面, mac 下我看不到 ssd 的温度, windosw 下能看到差不多是 30 度左右, 我是贴了散热片的, 装好跑完分就到 50 度了, 转接的可以考虑加个机箱风扇来减轻一下心理负担, 像我一样M.2 在 CPU 旁边的话, 讲道理空气流通还是可以的.(反正也看不到温度)，在主板背后的就随缘吧
  6. 其实我是随便的, 高速没什么感觉, 主要还是 iops 的提升, (真相: 看看靠谱的 SATA 如 三星850EVO涨价涨得飞起, 256 的也要将近 600, 不如上 sm951得了.)
  7. Sata 方面, 推荐一些便宜的选择, 英特尔Pro2500, 海力士拆机盘

键盘 Poker2

macOS 下 F 键没啥用了, 一直长草 61 键的迷你键盘, 茶轴有点累, 又非常想换MX红轴, poker2 重置版上了 jd, 价格也还可以, 每天都用的东西就不委屈自己, 咬牙剁了.

还是很值的, 红轴用起来相当不错, 省力不少, 但个人还是偏好更轻

原本想 Poker2 + MX Master 一步到位, 用一发 1000-200 的券, 后来看着这个陪伴我将近两年的笑脸, 想想算了.

无线网卡 BCM4322

  1. 认不清 mini 接口, 买错卡了, 24块钱的东西翻车了, 吃了没文化的亏啊. 后来买了个转接卡, 装上免驱到位了. 心好累.(事后用在了笔记本上)
  2. 加起来总共 45, 比一般的 69 是便宜了二十块. 带蓝牙的 269 想都不敢想, 虽然可以玩一玩 HandOff 这种没啥用的功能.
  3. B150 87V5啊, 我原本以为 “x4带宽的x16口” 和 “M.2” 是共享带宽的, 然而看了官方文档, 哦并不是. 这个还是不错的.
  4. 所以上 ITX 独显的同学如果不想用 USB 网卡, 就要选带 M.2 网卡口的啦.注意一般主板自带的英特尔无线网卡是无解的, 像我上面推荐的 技嘉B150N, 虽然自带的是英特尔, 但是可以换免驱网卡. 大多自带双英特尔网卡的都不能换. 要看仔细.
  5. USB 网卡挺好的, 就是驱动很难装上.

机箱 Segotep Mini 3

  1. 没选 ITX 迷你小机箱的残念, 选了个三围比较小的机箱, 关键是不要五百多, 不要三百多, 不要一百多, 只要69.
  2. 显卡长度就没办法了, 然而 250mm 可选的还是有一些.
  3. 质量真的很差…..机械硬盘震动相当感人，本来相当的安静
  4. 颜值方面原本是喜欢乔思伯UMX4黑色版的

内存 Kingston Fury DDR4 2133 8GB

  1. 刚要买, 颗粒就涨价了, DDR4 的价格翻了一番是怎么回事.
  2. 反正频率最高就 2133, 选了个时序还可以的 141414, CL12 的贵了六十, 果断选便宜的.
  3. 8G 确实不够, 等便宜了在加吧.

其他

电源风扇都直接选老大哥推荐的.
Dell 2515h, jd开车价 1999 确实不错, 25寸ppi略低, 推荐大家计算一下自己偏好的 ppi . 果断选了 1499 的 23.8寸 2k, 太乞丐了, 一个USB 都没有.
这里强行秀一下我的显示器支架.

驱动

核心显卡 HD530

  1. 10.12.1 对显卡支持挺好的, 这个核显也是套路深.
  2. 1080P 下, 什么都不用做就能用了, 假如你不介意系统信息显示成 Intel SKL Graphics, 也就是读不出型号.
  3. 2k 下就很坑了
    1. 用命令行强开 2k 毫无问题
    2. 右上角蜜汁闪屏, 显存只有 14m
    3. 打上 AppleIntelSKLGraphicsFramebuffer , SKL Alpha 等 n 个补丁和注入, 完成型号识别和显存识别
    4. 因为白苹果用的也是 hd530, 从那里扒来一个补丁修右上角闪屏.
  4. 最后显示成这样
  5. 当然之后还有 HDMI 音频问题, 忘了怎么弄好的了
  6. 性能方面, 1080p 或许真的能打 LOL, 2k下嘛, 亮机还是足够了的.

声卡

ALC892, 原理上有好几种方法驱动, 然后

  1. Voodoo 破音
  2. AppleHDApatcher , HDA 被驱动, 并没有声音
  3. 最后还是回到注入, 试来试去, AppleALC 换来换去, 是驱动了 AppleHDA.

SSD

NVMe 驱动补丁的迭代

  1. 10.11, 加入kext 作为外挂盘
  2. Sierra DP4 config补丁

  3. 正式版 Sierra config补丁.

USB

  1. 很麻烦，能用的话, 建议什么都不做
  2. 本来直接驱动，2.0认成3.0，3.0能插鼠标键盘网卡，唯独u盘不认。本来觉得这样也没什么，因为我就鼠标键盘，两个2.0偶尔用一用。后来查了查发现要改控制器，步骤很明确没什么难度, 就是烦, 而且是暂时的解决补丁, 添加100系主板通用补丁后，6个接口都能用了，但是都是100ms的读写, 我觉得已经很快了, 有需求在搞吧.
  3. 据说USB 不完美, 唤醒会有问题, 如上述不睡眠解决问题.
  4. 许多同款主板的同学表示插 3.0 装系统会有各种各样的问题, 我自己的话会 root卡住五国花屏

独显 不是我的AMD RX 480

  1. 之前就中了一个 影驰 1050Ti 4G….. 最终还是没用上他.
  2. 其实我个人相当喜欢这一代的 AMD 卡, RX 480 双11 开车价格也很低, 然而并没有上车(穷), 虽然据说在 macOS Sierra 10.12.1 里 480 已经显示成 AMD Radeon R9 xxx, 半驱动, 注入id之后完美驱动. 修改内核显示字符就能显示成480
  3. 驱动开始乱来了, id注入位置变来变去的, 10.12.2 彩虹花屏了, 既然白苹果用的 RX460, 应该是会一直有解的
  4. 各种蜜汁黑屏, 开机黑屏直到显示登录界面
  5. 顶级企业卖大楼, 真爱粉支持农企不倒闭.
  6. 注意, XFX黑狼虽然便宜料足, 但是很吵, 但是很吵, 但是很吵.
  7. Windows 驱动方面
    1. 蓝宝石迪兰Windows都会掉最近发布的鸡血驱动, 应该是通病.
    2. 叫Relife的驱动禁用了 GCN1.0 老卡的异步运算. 即便如此, GCN1.0 以上的新卡随着驱动更新在 Windows 下的表现越来越强,
    3. A卡战未来啊.

  8. Radeon 信仰灯就看看

内建功能

  1. 网卡: 放入驱动就 en0, iCloud AppStore 能用
  2. 白苹果三码注入, iMeassage 可以登录. Airdrop Handoff 没啥兴趣, 也没钱买带蓝牙的网卡.

分区布局

  1. 装macOS 装的时间比装 Windows 的时间少多了.
  2. 备份了一个 100G 的macOS 升级调驱动作死用
  3. Windows 的 UEFI 默认 EFI=100m, 然而 mac 磁盘工具需要 200m 才能运行. 真的尬.
  4. Windows10 LTSB 相当好用, 也许是原本个人就比较喜欢 Win10 的界面. 除了有好几次蓝屏, 相对之下 macOS 相当稳定
  5. 分出和ssd 大小一样的分区来做 Time Machine

总结

  1. 门槛还是很低的, 从一个纯小白, 到能简单装台主流机器, 当然少不了老大的一口大奶
  2. 11.1 jd价格的确跳水的厉害, AMD 显卡的车更是开个不停, 加上最近 CPU 的涨价, 差不多省了七八百吧. 虽然没有二手便宜.
  3. 听说双显卡可以一张显示, 一张计算, 核显显示, 独显计算, 这样的话还是不错的
  4. 全部算起来 6.25k, 确实不便宜了, CPU SSD 显示器, 三个加起来就 4k+ 了, 其他东西缩的还是很厉害的
  5. 没上 X99 =͟͟͞͞(¦3ꇤ[▓▓]. E5-2689v4 ES 价格也只能看看, 真心喜欢 2683v3.
  6. 从拍下到全部弄好, 正好一周. 总体来说玩的还是很开心的. 最大感想是 “原来现在的主流电脑已经这么好了” .
  7. 对于断舍离(扔)狂热爱好者, 全都用上才是值, 不费事省时间才是性价比, 马上投入工作才是生产力.
  8. 又不上白苹果，一分钱一分货(省个毛)

  9. 台式的最大优势还是性能的提升，所以结论就是，去他妈的家用配置，二手专业显卡，e5 走起。

未完结

第二步所有硬件驱动后，只是可用,
当然还有完善，也就是电源，睡眠，变频，内建，优先级如下

  1. 变频, 笔记本长期高频, 或者一直低频的情况
  2. 最麻烦的就是核显了, 花屏, 闪屏, 变频, 显存, HDMI, 都是问题
  3. 睡眠和唤醒, 首先你得能睡, 不会睡死, 再是唤醒后所有东西都正常
  4. 该加载的电源驱动最好也能加载
  5. 内建就随意了

我的台式机没动dsdt就全部可用了，(技嘉100系主板不支持原生电源管理, 反正我的核显唤不醒), 真正用到dsdt的，是我最近在闲鱼上淘的二手笔记本

方案

  1. X79 + i7 - 3690X
    12框不多, 板还难找, CPU 是不错, Pass
  2. X99+ E5 - 2683v3
    14C28T, 相当好的方案, 没选他其实有点小后悔. 其实还是因为预算, 双路上不起, 单路板太贵了. 此外显卡必买, 原本打算慢慢攒.
  3. B150M + i7 - 6700
    因为这一代i3i5i7 核显都是 hd530, i5也升级到了物理四核, 选 i7 是免得想升级七代. 后面再来详细说这个蜜汁核显. 办公上网 i3 确实足够了, 大部分场景我的 CPU 都是35摄氏度, 基本空载, 但以后笔记本都买个 i7 性能比桌面的还好, 岂不是尴尬.
  4. P8Z77M + E3 - 1230V2/i7-3770k
    加起来就 1500 以下, H61有新的也不便宜, 而且不能用 PCIe SSD, Z77m 只能选看上去靠谱的二手, 华硕 matx 小板规格强无敌,
    1. 支持双 x8 PCIe 拆分, 一条显卡, 一条转接 PCIe ssd 直走CPU
    2. 老主板改 BIOS, 刷 OZ 一包入魂, 直接洗成白苹果
    3. E3 的 DSDT 不知道多到哪里去了. 随便拿来用就好了.
    4. 关键还是靠谱Z77难找
      入手价格比较好的 Z77 也是不错的, OZ 有现成完美文件, 并且能支持 双x8 的华硕型号如下, 右下 OZ 官网.

ITX

  1. 技嘉B150N + SKL
  2. ITX 迷你小机箱, 布局紧凑又方便带, 也不占地方, 办公机可以算首选了
  3. 这个板接口很多, x16独显, M.2 PCIeSSD, M.2笔记本网卡接口, 十分适合
  4. 带英特尔无线网卡版899, 不带网卡699, 价格也还可以接受

3.4 两个差 七八百, 最终还是选了新平台,

  1. 保修靠谱, 避开许多老平台的问题
  2. Clover 麻烦点就麻烦点, 好多东西都发生了改动, Clover 支持还是非常快的
  3. 自带核显亮机,
  4. 原生 M.2, 虽然走主板带宽, 但速度还可以
