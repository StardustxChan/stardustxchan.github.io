商品 ADC 面试总结

按照时间线记录了在职的几次面试. 真实招聘.
Attack Damage Carry

8.17: 面试 MLi

  1. pdca = 完整的商品企划本该如此, 但是我回答的范围太大了，现有职位不要求这些. 和我提实际有落差, 我也知道啊, 企划 = 1数据分析，2规划，3验证下店，4思考可选
  2. 非要我选一个商企特长, 没想过, 因此回答了人群, 感觉太热, 紧张发挥不好, 太针对人群了，最后应该补上 “潮流落地到人群的消费购买”. 其实正确的回答是 “数据”
  3. 讲太多感性反而不理科, 用力过猛了, 我应该说这些因素的分析都需要数据, 感性是确保能收集到, 关注不代表精通.
  4. 说我讲的太大了，工作反而局限, 所以最后主动说我觉得我还算开放, HR 也承认了我比较开放, 所以才分问题询问我每个方向的问题
  5. 推荐: 陈列，直营AD， 招数学和计算机的数据分析, 考虑能给我新增岗位是 “店铺分析”.
  6. HR 说能产生什么价值, 这是我应该前期准备的问题. 瞎回答了发掘遗漏, 被怼了查漏补缺得要先懂整个过程. 只好说能带来新的角度.
  7. 说我简历死板纯粹，做人纠结不自信, 表达太注重正确性, 连用”可能”. 聊了才觉得我有创新, 有想法, 高领悟交流容易
  8. 总结: 整体流程即把面试者按照回答标签化, 然后匹配岗位. 容易由一个没想过的问题回答一个不太能准确描述自我的答案而让谈话进入异常. 应对方法依旧是状态好的话一步三算, 以及回答的时候也回答对立面, 更全面.

二次面试, 真实社招

看到一帆姐旧旧的工牌上写着商品专员, 一直到新上任总监. 有些感慨是不是我坚持不够, 但是转念一想.

Re: MLi

  1. 整天 pcie 的研究电脑, pdca 讲成 pcda, 被说了.
    1. PlanDoCheckAct, 无聊的缩写知识点.
    2. HR 讲起这个嘴角难掩的明显笑容将会成为我的心理阴影,
    3. “你还是要学习, 这次在我面前错了没关系, 你以后如果要面试就不能错.”
  2. 飘了所以什么话都敢讲了
    1. 聊到我觉得纳纹的企划除了不招人以外, 有什么不合适我的, 正确回答大概是调性不合我个人
    2. 但是我就偏偏回答, 讲到由于老板娘和老总们具有决定权, 所以企划全程会无意间朝着老板娘的口味偏. 我也没说老板娘这样不好, 误导了. HR 理解成这是我不想做纳纹商企的唯一原因了.
    3. “所以你是觉得 NONE 受老板娘的控制小, 自由发挥的范围大, 是吗?”, 我开这么多会并不是在睡觉. “你未必有老板娘懂消费者和市场”
    4. 回答不是就一直问, 就回答是.
  3. 讲错话而被过度推断, 其实并不冤枉.
  4. 问我为什么皱着眉头讲话, 只好回答感冒了喉咙难受, 没别的意思.
  5. 感觉自己还是不会讲话, 最后只有强行自信微笑.
  6. 最后把 MUJI 的 Keynote 当做所谓的敲门砖发给了 NONE 企划专员, 过了第一关还行.

招一个脚踏实地做基本工作的管培生, 方向错了

  1. 感觉专员很累, 根本不想听我讲话, 我还使劲比比. 看着我两个月前写的简历.
  2. 问我萧山人是不是不愿意承认杭州, 开玩笑解释了一手
  3. 实习经验, 嗯, 没有服装行业的经验.
  4. 我觉得商品企划是什么, 第二次回答这个问题
    1. 先搞懂自己是谁, 要做什么, 不要因为专业素养的直觉反应而误导最初的目的
    2. 注意用户的需求, 理念高于需求, 就引导消费者或者思考是不是自己错了. 需求高于理念, 就提升服务和产品.
    3. 最后才是研发和正常走流程
    4. 正确回答是, 管培生应该抱着学习的态度, 在极小视野下发表一些专员看来他们入行时也这么认为的看法, 同时展示自我的可发展性, 重点表达对上级的服从.
  5. 问我觉得纳纹好看吗喜欢吗
    1. 必须回他妈的避, 拒绝两头不是人
    2. 回答这是用户需求的结果, 陶总早就想的很明白了
  6. 是否了解设计师品牌
    1. NONE 是自己养了一波设计师当做自营, 应该就是 UD hits 了, 还有就是, 专门买手拨预算去买设计师品牌的某几样产品
    2. 我的回答如下,
      1. 贵, 用来筛选消费者
      2. 设计风格固定, 主题不固定
      3. 胆子大, 敢, 新事物在流行前都是特立独行的
    3. 总结: 环境允许消费者表现自我, 途径互联网满足信息传递(这里应该回答圈子替代, 因为 NONE 没有网店), 终端消费者买得起.
    4. 最后, 诚实表达我没买过, 她回答很正常, 现在想来这个回答不错, 避免了我风格口味和 NONE 很不相符.
  7. 举一个完整的用户体验方案, 想了一会选择了大二的调研 “制服诱惑”, 目的是表现我的创意
    1. 目的: 让同学们了解一下自己和别人的想法
    2. 途径: 图书馆发他妈的问卷, 和网络结合, 不同途径的答案自然有不同
    3. 问卷
      1. 惯例回答前后问题的验证, 不是预设结论的顺序题目罗列, 而是拎出几题是相互验证的题来深入分析用户的想法.
      2. 复杂句型问题问深思熟虑结果, 简单直接句型问直觉反应
      3. 次序对答案的影响, 打乱顺序成三个版本
      4. 场景对答案的影响, 不该在用户忙的时候问具体的心情, 这样是无效的.
      5. 最后第 16 题, 留给发问卷的同学, 满足恶趣味的提问, 最后作为演讲彩蛋. 实际上这个可以改成上一个答题者给下一个答题者出一道题.
    4. 总计时间回答了 20-30h
    5. 其中还穿插的强行拉到个人能力的几句话忘了.
  8. 我问新来的妹子的工作
    1. 街拍市调还没带他去
    2. 最主要是更新竞品
    3. 微信九宫格和推送
  9. 我问专员的工作
    1. 取决于我来了之后, 估计是负责带人的.
    2. 在追问, 回答太多了, 估计是做了很多杂事和难言之隐, 注意力都在想问题, 无法仔细观察了.
  10. 她对我说的话
    1. 要下 2-3 个月店
    2. 精神和物理上辛苦, 我说接受8106, 他说没这么苦, 我说反而有点那个, 迅速跳过话题
    3. 说我讲的层次太高, 能不能脚踏实地, 我迟疑了下回答能. 她果然问我为什么, 回答理科生的毛病是不确定的事不下结论, 刚在想究竟要做些什么
    4. 数据分析和 Excel 就说我是理科生. 现在回想, 她说 “至少比文科生强”, 不能接着她说, 因为她可能就是文科生.
  11. 多次作死证明, 面试什么岗位, 就是什么技能和性格.

8.18 管培生会议

开局和结束，鞠躬的只有我一个人

把自我比作 mbp 的自我介绍

  1. 做事方面: 配合绚丽屏幕视频剪辑性能强, 文字工作专注的效率第一, 对于某件事, 我是否要使用优势做取舍
  2. 目标方面: 大电池能用很久. 是因为内部空间紧凑设计, 百年时间, 把最多的时间留在核心重要的事上
  3. 做人姿态: 只有一种最新的雷电接口, 因为舍弃了面面俱到的 USB 和网线, 才能通过这种高速接口连接显卡提升性能. 为了核心目的有取舍.有些不可能的事之所以不可能, 也许是因为放不下
  4. 最后, 表面的触摸条, 指纹这些酷炫功能我没说, 我可以罗列我的特点，但是这些酷炫的会被更新取代，唯有理念延续，侧面反映了我在有限的时间内的取舍

临时的代表发言

  1. 感谢x总能给我这个机会代表管培生发言, 和刚才两位同事的情况不同, 我是临时的发言, 因此没有做准备, 只是讲一些真实的感受和想法.
  2. 来到我们纳纹品牌的商品中心实习已经将近两个月, 给我最大的感受就是各位前辈工作很认真负责, 都是专注实干的人. 虽然不同的部门之间会有矛盾和冲突, 但是大家的出发点都是为了让纳纹这个品牌能够更好, 这是不同部门之间统一的目标.
  3. 在商品中心也有很多学习的机会, 审样会和 19春订货会我都有参与其中, 前辈和上司都很关注新人的学习, 特别感谢我们商品中心的总监姐和负责带我的自联营区的姐, 还有我的直属上司. 她们工作很努力, 也教了我很多, 我觉得像她们那样就算的上称职的员工. 我想各位管培生应该也得到了许多这样的机会, 也有需要感谢的人.
  4. 这样的工作氛围很温暖, 我说的也包括身体上的, 因为我刚来的时候空调不是很给力, 有点热, 当然现在已经好了. 不过不是因为我今天穿成这样才热, 今天我多穿是因为我感冒了.
  5. 与商品中心的前辈们的工作相比, 我其实没帮上什么忙, 还需要时间继续学习. 可能不像你们那样, 帮了前辈很多.
  6. 今天唯一的遗憾是x总没来, 我很喜欢宋总, 她是我的女神.
  7. 还有就是公司福利很好, 给分配员工宿舍, 前几天还有房东帮忙搬家, 嗯, 还有就是公司附近的外卖很好吃
  8. 最后感谢 HR 能看得起我招我进公司, 谢谢大家.
