陈列培训

在 NA 陈列培训的一整天: 理论知识 + 下店操作
主要记录了知识点, 以及陈列师视角的看法.

理念

核心理念: 从摆货到体验

A 品牌形象: 流通型精品集成品牌店 (态度明确, 风格明细, 定位精准明确)
B 货架构造: 移动, 墙面, 落地, 吊挂
C 特色氛围区

卖场陈列理念:

  1. 打造小而巧的摩登, 浪漫, 精致, 风情的时髦衣橱
  2. 打造非凡的视觉体验和消费, 逛一逛都是一种享受
  3. 生活化的时髦, 精致, 浪漫, 风情女性气质

陈列模式

  1. 突显单个系列产品风格的单间陈列
  2. 卖场整体的色系过度
  3. 氛围区的独特, 新颖, 形成体验式消费 (橱窗, 展桌, 矮柜, 吊挂杆, 陈列梯, 板层贵)

店铺布置

新品到店的工作流

  1. 放黄金区
  2. 入口区
  3. 模特
  4. 过季规划

模特

  1. 橱窗 1.85m, 店内 1.75m
  2. 有头无头结合, 朝向, 一前一后一高一低,
  3. 遵从整体的无脑三角构图

橱窗: 展示精华

  1. 意义: 展示理念, 态度, 趋势, 产品即时信息
  2. 激发: 好气, 理念认同, 风格, 信息

点挂

操作

  1. SM 两件出样
  2. 左三分之一: 点挂 2 件
  3. 中间挂帽子和包
  4. 右三分之一: 当成侧杆挂衣服, 下面放鞋子

点挂原则

  1. 按系列
  2. 按单件
  3. M
  4. 颜色深浅间隔
  5. 风格相同
  6. 吊牌放在理念

陈列形式

  • 1/1/1: 单款单色成套, 颜色循环
  1. 最佳搭配, 适用于 SKC 多
  2. 整体颜色如果暗, 亮色就双件出样
  • 1/2/1: 散件, 颜色一致
  1. SKC 少, 种类也少
  2. ‘2’ 用相同面料或者相同颜色

一杆的衣服数量

  1. 1.2m: 12-14件春夏, 10-12 件秋冬
  2. 1.6m: 14-16件春夏, 12-14 件秋冬

展柜

展柜从上到下:

  1. 相框, 头模
  2. 包, 首饰
  3. 鞋子
  4. 书

展桌用法

  1. 整合可用的道具
  2. 分组
  3. 强行三角构图

龙湖天街下店

总体策略

  1. 主客流动线, 客户从哪一边进来
  2. 观察对面其他品牌的主色系，品类
  3. 一杆陈列少或者没做完会给消费者缺货的感觉，没有看得必要
  4. 门口放的引流款，对于不同的款, 过来摸的人群有年龄分层

实际陈列

  1. 模特的动作到位或者什么动作都不做
  2. 门口的桌子上的裤子, 平摊开上放鞋子, 或者卷起来
  3. 挂着的长裙下不能放花瓶，会碎
  4. 衣架朝向一致
  5. 下搭不能挂边上，正反看裤子不能在衣服前面,
  6. 一杆衣服包含: 1件正挂，8件侧挂，品牌还算是便宜, 所以可以多一点

实际体验

  1. 针对 [总体策略1], 用户体验不是从进店开始, 而是从看到店面开始, 因此要模拟用户从不同的位置走过时的视角
  2. 店铺通透性与可视角, 有凸显调性的盆栽在视觉上挡住了主推款
  3. 从左往右的人群: 店面昏暗导致几乎所有客户都是先看对面的门面, 看完转头时已走到正门口, 配合视线遮挡十分不利
  4. 从右往左的人群: 另一边没有店, 因此店铺右边的货杆被主推, 看得人最多得到解释
