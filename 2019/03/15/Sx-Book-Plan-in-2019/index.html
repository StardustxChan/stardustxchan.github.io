Sx-Book-Plan in 2019

之前有学习如何学习(Learning How To Learn), 这篇是阅读如何阅读(Reading How To Read).
希望大家能通过了解我的方法, 来更好的形成自己的阅读体系.

精简目录

  • EDC Message: 只记住有用的 1% 信息每天携带
  • 小说消遣与电影无异: 摘录转折点和坚持点 + 我做不到的, 结果导向全局分析
  • 空白页自建 MindMap, 是因为用软件会自带分解属性理解知识, 希望零散知识也能成网流通, 互相解释.
  1. The Key: 他人按照目录建立的思维导图无法记住, 且网上随处可见, 框架有了如何取用 Key, 也就是 ‘搜索提取知识’ 上框架并不擅长
  2. EDC Message 方法论: 思考过程才能进步
    1. 摘录对 Sx 有用的的片段
    2. 断舍离删除无用, 对于 ‘已吸收的内容’, Review 时你会认为这不是废话吗的本该如此
    3. ReBuild, 无视目录重构全书架构, 做一个 Sx 的 OTA 更新包
    4. Review二八法则: 充分吸收融入 EDC Message, 形成记忆提取点 The Key
  3. 为我所用, 读什么 和 如何利用时间
  4. 计划和坚持, 目的几何, 要成为怎样的自己, 没有计划时间就会被无意义信息占据
  5. 再来选择书

1. 我们是为什么在读书?

看到这个标题, 你有什么感受.
迫不及待的几种回答如: ‘知识’, ‘人生高度’, ‘和别人不同’

我自己的回答仅仅是: ‘我喜欢, 或者我愿意把这段时间消费在读书上.’
由于没有那么强的目的性, 我并不期望吃透书, 不期望理解架构, 不期望谈笑风生. 仅仅是像看一场电影一样, 下了花时间且做好 ‘零收益’ 的准备(如果你非要和努力的别家孩子对比, 你的收益还可能是负的), 投入当中, 去经历, 去做没有限制的思考, 以及写这种 ‘无干货, 不鸡汤’ 的总结.

2. 读完书到底收益了什么?

我的第二个回答是: ‘

  • 就像电影, 你收益了快乐的一段时间, 你获得了经历过这段时间后的自己.
  • 这段时间将对你的未来产生或多或少的影响, 就像人生中已度过的任意一段时间.’

EDC = every day carry, 是指每日携带
EDC Message 是我生造的词, 指 ‘这本书对你微小的改变, 或许是记住某一句话, 你将这一点信息理解, 做任何其他不相关的事时, 也能想起它, 甚至用到它.’

EDC Message: 只记住有用的 1% 信息每天携带

因此这个信息必然是高度定制化, 形式是一张网, 由书中原文作为原点(The Key)出发, 连接你的经历, 你的想法, 你曾看过的别的事物, 等等. 微信阅读上, 有几百上千个人 mark 了同一句话, 但是他们各自有各自的经历和理解, 以及各自不同的遗忘曲线.

3. 记笔记杂谈, 非要提高收益效率的话, 有什么思路?

3.1 大脑在存取时的特色

我倾向认为, 人类大脑记忆是通过关键词(The Key)随机存取的, 无论把课文或者大纲建立的多么清楚, 该记住的只有最深刻的那么一点, 这一点往往是读者自己的故事.

记笔记: 抄书也好, 用思维导图分好目录再抄也好, 过几个月连 EDC Message 都不一定记得. 除非当中有非说不可的 ‘自己的思考’, 也就是你自己的故事.

3.2 思维导图杂谈

对于按目录分的思维导图, 过几天会记不住目录, 由于记不住目录, 目录之下的 ‘三级信息’ 就很难通过树从上至下搜索到, 所以往往最终搜索信息的方式还是通过关键词跨级随机读取.

思维导图用来回忆是很好的, 因为链接是你自己建立的, 即便过几个月你会忘记连接的两头, 但是创建连接的方法是可以复用的.

3.3 下面分享一下我面向 EDC Message 的笔记法:

  1. 摘录片段, 直至看完全书.
  2. 断舍离删除无用片段, 对于 ‘已吸收的内容’, Review 时你会认为这不是废话吗的本该如此
  3. ReBuild, 无视目录重构全书架构, 留下的片段是很少的, 排序形成主题易于理解和回忆.
  4. Review二八法则: 充分吸收融入 EDC Message, 形成记忆提取点 The Key
  5. 最后往往只剩下几句话, 我将会在未来一段时间内, 忘掉其中 80% 以上

4. 最重要的 ‘选择书’ 和 ‘分配时间’ 问题

由于高度个人化, 任何攻略都不如自己总结经验, 自己想的即便是错的也愿意尝试.

  1. 兴趣: 希望你看了我这么随便的读书目的能够不那么焦虑. 兴趣是最大的驱动力, 珍惜兴趣并且好好保护和培养, 它让我们以孩童的眼光看新的事物.
  2. 时间分配: 还是免不了说, 没有计划时间就会被无意义信息占据, 坚持做自己感兴趣的事是一种幸福.
  3. 选择书: 要成为怎样的自己, Fake it till You Make it, 其实只要不被绑架, 这里的选择也不局限于书.

5. 总结

最后适合用一句名言来结束全文.

‘早点睡觉.’ —–by 鲁迅
